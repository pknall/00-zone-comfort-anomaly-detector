### Zone Comfort Anomaly Detector

---

Generate a prioritized list of zone anomalies for the user to investigate.

Read in the Zone Temperature, Heating Setpoint, and Cooling Setpoint for a collection
of zones and return a prioritized list of any zones that exhibit anomalous 
behavior.  

---

#### Dependencies
* Python v3.14
* psycopg2 v2.9.11
